# クラスターエディタ

クラスターエディタは、AI が検出した入力項目（クラスター）を視覚的に編集するための画面です。**選択モード** と **作成モード** の2つのモードがあります。

![クラスターエディタ全体](/screenshots/editor-overview.png)

---

## モード切替

ツールバー左端のボタン、またはキーボードショートカットでモードを切り替えます。

| モード | ボタン | ショートカット | 用途 |
|--------|--------|----------------|------|
| 選択 | 矢印アイコン | `V` | クラスターの選択・移動・リサイズ |
| 作成 | ＋アイコン | `N` | 新規クラスターの追加 |

`Esc` キーでいつでも選択モードに戻れます。

---

## 選択モード

### 単一選択

クラスターをクリックすると選択されます。選択中のクラスターは青い枠線で強調表示され、右側のプロパティパネルに詳細が表示されます。

### 複数選択

| 操作 | 方法 |
|------|------|
| 個別追加/除外 | `Shift` または `Ctrl`/`Cmd` + クリック |
| 範囲選択 | 背景をドラッグして矩形で囲む（ラバーバンド選択） |
| 全選択 | ツールバーの「全選択」ボタン |
| 選択解除 | 背景をクリック、またはツールバーの「選択解除」 |

![ラバーバンド選択](/screenshots/rubber-band.png)

### 移動

**ドラッグ移動:**
- 単一選択: クラスター内部をドラッグ
- 複数選択: バウンディングボックス（破線の囲み枠）をドラッグすると全選択クラスターが一緒に移動

![グループ移動](/screenshots/group-move.png)

**キーボードナッジ:**

| キー | 移動量 |
|------|--------|
| `↑` `↓` `←` `→` | 1px |
| `Shift` + 矢印 | 10px |

ツールバーのナッジボタン（十字矢印）でも同様の操作が可能です。

### リサイズ

単一選択時に表示される 8 つのハンドル（角 + 辺の中央）をドラッグしてサイズを変更できます。

### 削除

- 選択中にツールバーの「削除」ボタン
- 右クリックメニューから「削除」

---

## 作成モード

AI が検出しなかった入力項目を手動で追加する機能です。

### 操作手順

1. ツールバーの「作成」ボタンまたは `N` キーで作成モードに切替
2. カーソルが十字（crosshair）に変わります
3. プレビュー上でドラッグして矩形を描画

![作成モードドラッグ](/screenshots/create-drag.png)

4. マウスを離すとポップオーバーが表示されます

![作成ポップオーバー](/screenshots/create-popover.png)

### AI 推論

描画した矩形内のセル情報を AI が自動分析し、以下を推論します:

- **名前**: クラスター名（周辺ラベルから推測）
- **タイプ**: 入力項目の種類（テキスト、数値、日付など）

推論結果はポップオーバーに自動入力されますが、自由に編集できます。

### 確認と作成

- 名前・タイプを確認/編集して「作成」ボタンをクリック
- 作成後は自動的に選択状態になります
- **作成モードは維持されます** — 続けて次のクラスターを描画できます

::: tip AI をスキップ
AI 推論を待たずに「スキップして手動作成」リンクで即座に手動入力に切り替えられます。
:::

## ズーム操作

| 操作 | 方法 |
|------|------|
| ズームイン | ツールバーの虫眼鏡(+)ボタン |
| ズームアウト | ツールバーの虫眼鏡(-)ボタン |
| ズームリセット | パーセンテージ表示をクリック |
| マウスホイール | `Ctrl`/`Cmd` + スクロール |

ズーム範囲: 50% 〜 300%

---

## プロパティパネル

クラスターを選択すると、右側のプロパティパネルに以下の情報が表示・編集できます:

- **セルアドレス**: Excel 上の位置 (例: C5)
- **信頼度**: AI の推測確度
- **名前**: クラスター名
- **タイプ**: 入力項目の種類
- **読み取り専用**: トグル
- **パラメータ**: 入力パラメータ (key=value 形式)

---

## キーボードショートカット一覧

| キー | 操作 |
|------|------|
| `V` | 選択モードに切替 |
| `N` | 作成モードに切替 |
| `Esc` | 選択モードに戻る / 操作キャンセル |
| `↑` `↓` `←` `→` | 選択クラスターを 1px 移動 |
| `Shift` + 矢印 | 選択クラスターを 10px 移動 |
| `Ctrl`/`Cmd` + スクロール | ズーム |
| `Shift` + クリック | 複数選択に追加/除外 |
| `Ctrl`/`Cmd` + クリック | 複数選択に追加/除外 |
