# 基本フロー

Next i-Reporter は **4ステップ** で Excel 帳票を ConMas i-Reporter XML に変換します。

## 全体の流れ

```
Excel アップロード → プレビュー確認 → AI 解析 → 編集 & XML ダウンロード
```

---

## Step 1: Excel アップロード

トップ画面で `.xlsx` ファイルをドラッグ&ドロップ、またはクリックして選択します。

![アップロード画面](/screenshots/upload.png)

::: tip 対応形式
`.xlsx` (Excel 2007以降) のみ対応しています。`.xls` は非対応です。
:::

---

## Step 2: プレビュー確認

アップロードされた Excel のシート構造がプレビュー表示されます。

![プレビュー画面](/screenshots/preview.png)

- セルの値、結合、スタイルが確認できます
- 複数シートがある場合はタブで切替可能
- この段階で内容に問題がなければ「AI で解析する」ボタンを押します

---

## Step 3: AI 解析

OpenAI GPT がセル構造を分析し、入力項目（クラスター）を自動検出します。

解析中はローディング画面が表示されます。通常 10〜30 秒で完了します。

**AI が検出するクラスタータイプ:**

| タイプ | 説明 | 例 |
|--------|------|-----|
| テキスト入力 | 文字を入力するセル | 氏名、住所、備考 |
| 数値入力 | 数値を入力するセル | 温度、数量、金額 |
| 日付 | 日付を入力するセル | 実施日、記入日 |
| 時刻 | 時刻を入力するセル | 開始時刻、終了時刻 |
| 計算式 | Excel 数式があるセル | 合計、小計 |
| 選択 | ドロップダウン選択 | 区分、ステータス |
| チェック | チェックボックス | 確認済み、合否 |
| 画像 | 写真エリア | 現場写真 |
| 手書き | 署名・印鑑エリア | サイン欄 |

---

## Step 4: 編集 & XML ダウンロード

AI 解析結果がクラスターエディタに表示されます。

![クラスターエディタ](/screenshots/editor-overview.png)

- 左側: PDF/テーブルプレビュー上にクラスターがオーバーレイ表示
- 右側: 選択中クラスターのプロパティパネル

### 編集操作

クラスターの位置・サイズ・タイプを自由に調整できます。詳しくは [クラスターエディタ](./cluster-editor) を参照してください。

### XML ダウンロード

編集が完了したら「XML を生成」ボタンをクリックします。生成された XML ファイルは ConMas i-Reporter にそのままインポートできます。
