# XML インポート

既存の ConMas i-Reporter XML ファイルを読み込んで、クラスターエディタで再編集できます。

---

## 対応する XML

ConMas i-Reporter Designer で書き出した **帳票テンプレート XML** に対応しています。

::: info ファイルサイズ上限
最大 **50MB** まで読み込み可能です。
:::

---

## インポート手順

### 1. ファイルをアップロード

トップ画面のアップロードエリアに `.xml` ファイルをドラッグ&ドロップ、またはクリックして選択します。

Excel (`.xlsx`) と同じアップロードエリアですが、拡張子 `.xml` を検出すると自動的にインポートモードに切り替わります。

### 2. 自動パース

XML の構造が解析され、以下の情報が復元されます:

| 項目 | 内容 |
|------|------|
| シート構造 | セクション・ページ構成 |
| クラスター | 名前・タイプ・位置・サイズ |
| パラメータ | 各クラスターの inputParameters |
| 読み取り専用 | ReadOnly フラグ |

### 3. クラスターエディタで編集

インポートが完了すると、AI 解析後と同じクラスターエディタ画面が表示されます。Excel アップロード時と同様に、すべての編集操作が利用できます。

---

## 使いどころ

| シーン | 説明 |
|--------|------|
| **既存帳票の修正** | Designer で作った XML を微調整したいとき |
| **テンプレートの再利用** | 既存帳票をベースに新しい帳票を作りたいとき |
| **AI 解析結果の確認** | 以前エクスポートした XML を再読み込みして確認 |

---

## Excel フローとの違い

| | Excel アップロード | XML インポート |
|---|---|---|
| AI 解析 | 必要（自動実行） | 不要（クラスター情報が XML に含まれる） |
| プレビュー画面 | あり | スキップ（直接エディタへ） |
| PDF プレビュー | Excel から生成 | XML 内の PDF データを使用 |

::: tip Excel → XML → 再インポートも可能
一度 XML をダウンロードした後、再度インポートして編集を続けることもできます。作業を中断・再開する際に便利です。
:::
