# よくある質問 (FAQ)

---

## アップロード・ファイル関連

### Q: どのファイル形式に対応していますか？

**Excel アップロード:** `.xlsx`（Excel 2007 以降）に対応しています。`.xls`（Excel 97-2003）は非対応です。

**XML インポート:** ConMas i-Reporter Designer で書き出した `.xml` ファイルに対応しています。

### Q: ファイルサイズの制限はありますか？

| ファイル種別 | 上限 |
|-------------|------|
| Excel (.xlsx) | 10MB |
| XML (.xml) | 50MB |

### Q: 複数シートの Excel に対応していますか？

はい。プレビュー画面でシートタブを切り替えられます。AI 解析は表示中のシートに対して実行されます。

## AI 解析関連

### Q: AI 解析にはどのくらい時間がかかりますか？

通常 **10〜30 秒** です。セル数が多い帳票や複雑なレイアウトの場合はもう少し時間がかかることがあります。

### Q: AI が検出できるクラスタータイプは？

現在 AI が自動検出対象としているのは以下の 10 タイプです:

KeyboardText, FixedText, Date, Time, InputNumeric, Calculate, Select, Check, Image, Handwriting

全 38 タイプの詳細は [クラスタータイプ一覧](/reference/cluster-types) を参照してください。AI 未対応のタイプは、クラスターエディタの作成モードで手動追加できます。

### Q: AI の解析精度を上げるにはどうすればいいですか？

- **用紙サイズと向き** を正しく設定してからAI解析を実行してください
- **セルの結合** が適切に設定されている Excel のほうが精度が高くなります
- **データバリデーション**（ドロップダウンリスト）が設定されていると、Select タイプの検出精度が向上します
- **数式セル** があると Calculate タイプが自動検出されます

### Q: AI が検出しなかった項目を追加するには？

クラスターエディタの **作成モード** (`N` キー) に切り替えて、プレビュー上でドラッグして矩形を描画します。詳しくは [クラスターエディタ > 作成モード](/guide/cluster-editor#作成モード) を参照してください。

## クラスターエディタ関連

### Q: クラスターの位置がずれている場合は？

以下の方法で調整できます:

1. **ドラッグ移動** — クラスターを直接ドラッグ
2. **キーボードナッジ** — 矢印キーで 1px 単位の微調整（Shift + 矢印で 10px）
3. **ツールバーのナッジボタン** — 十字矢印ボタンで移動

### Q: 複数のクラスターを一度に移動するには？

1. `Shift` + クリックまたは `Ctrl` + クリックで複数選択
2. または背景をドラッグしてラバーバンド選択
3. バウンディングボックス（破線枠）をドラッグで一括移動

### Q: クラスターのタイプを変更するには？

クラスターを選択すると右側のプロパティパネルが表示されます。「タイプ」のドロップダウンから変更できます。タイプを変更するとパラメータがそのタイプのデフォルト値にリセットされます。

### Q: 信頼度（Confidence）とは何ですか？

AI がそのクラスターの検出結果にどれだけ確信を持っているかを 0〜1 の数値で表したものです。

| 範囲 | 意味 | 推奨アクション |
|------|------|----------------|
| 0.95 | 非常に高い | そのまま使用可能 |
| 0.85〜0.95 | 高い | 軽く確認 |
| 0.70〜0.85 | 中程度 | タイプとパラメータを確認 |
| 0.50〜0.70 | 低い | 要確認・修正 |

ツールバーのフィルターで信頼度別に表示を絞り込めます。

## XML 出力関連

### Q: 生成された XML はどこで使えますか？

**ConMas i-Reporter** にそのままインポートできます。Designer でテンプレートとして読み込んでください。

### Q: XML を生成した後に修正したい場合は？

2つの方法があります:

1. **ブラウザ上で続行** — エディタに戻って修正し、再度 XML を生成
2. **XML を再インポート** — ダウンロードした XML をもう一度アップロードして編集を再開（[XML インポート](/guide/xml-import) 参照）

## 環境・セットアップ関連

### Q: OpenAI API Key はどこで取得できますか？

[OpenAI Platform](https://platform.openai.com/) でアカウントを作成し、API Keys ページからキーを発行してください。GPT-5.1 対応のキーが必要です。

### Q: API の利用料金は？

OpenAI の従量課金が適用されます。1 帳票あたりの解析コストは帳票のセル数によりますが、通常は数円〜数十円程度です。

### Q: ブラウザの推奨環境は？

| ブラウザ | 対応 |
|----------|------|
| Chrome (最新版) | 推奨 |
| Edge (最新版) | 推奨 |
| Firefox (最新版) | 対応 |
| Safari | 未検証 |

## トラブルシューティング

### Excel のアップロードが失敗する

- ファイルサイズが **10MB 以下** か確認してください
- `.xlsx` 形式であることを確認してください（`.xls` は非対応）
- ファイルが破損していないか、Excel で正常に開けるか確認してください

### AI 解析がエラーになる

- `.env.local` に `OPENAI_API_KEY` が正しく設定されているか確認してください
- API キーの利用制限（Rate Limit）に達していないか確認してください
- ネットワーク接続を確認してください

### クラスターが表示されない

- ツールバーのフィルターが有効になっていないか確認してください
- タイプフィルターや信頼度フィルターをリセットしてみてください
- ズームレベルを調整して、クラスターが画面外に出ていないか確認してください

### XML のインポートが失敗する

- ConMas i-Reporter 形式の XML であることを確認してください
- ファイルサイズが **50MB 以下** か確認してください
- XML が破損していないか、テキストエディタで開いて構造を確認してください
