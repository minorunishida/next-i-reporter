# クラスタータイプ一覧

ConMas i-Reporter で使用できる全 **38 種類** のクラスタータイプです。カテゴリごとに分類しています。

::: tip AI 自動検出対象
★ マーク付きのタイプは AI 解析で自動検出されます。それ以外はクラスターエディタの作成モードで手動追加できます。
:::

---

## テキスト

テキスト入力に関するタイプです。

| タイプ名 | 日本語名 | 値 | AI | 用途 |
|----------|----------|-----|-----|------|
| KeyboardText | キーボードテキスト | 30 | ★ | キーボードで文字を入力する一般的なテキスト欄 |
| Handwriting | 手書きデジタル | 119 | ★ | 手書き文字をデジタル変換して入力（署名など） |
| FixedText | 手書きノート形式 | 20 | ★ | 手書き入力をそのまま画像として保存 |
| FreeText | 手書きフリーメモ | 10 | | ホワイトボード風のフリーメモ |

**よくある使い分け:**
- 氏名・住所・備考 → **KeyboardText**
- 署名・印鑑欄 → **Handwriting**
- 手書きのスケッチやメモ → **FixedText** / **FreeText**

---

## 数値・計算

数値入力と計算に関するタイプです。

| タイプ名 | 日本語名 | 値 | AI | 用途 |
|----------|----------|-----|-----|------|
| Numeric | 数値選択 | 60 | | スライダーやステッパーで数値を選択 |
| InputNumeric | 数値 | 65 | ★ | テンキーで数値を直接入力 |
| NumberHours | 時間数 | 110 | | 時間数（○時間○分）の入力 |
| Calculate | 計算式 | 67 | ★ | 他クラスターの値を参照して自動計算 |

**よくある使い分け:**
- 温度・数量・金額 → **InputNumeric**
- 合計・小計 → **Calculate**（Excel の数式セルから自動検出）
- 作業時間 → **NumberHours**

---

## 日付・時刻

日付と時刻の入力に関するタイプです。

| タイプ名 | 日本語名 | 値 | AI | 用途 |
|----------|----------|-----|-----|------|
| Date | 年月日 | 40 | ★ | ドラムロール式の日付入力 |
| CalendarDate | カレンダー年月日 | 111 | | カレンダー UI から日付を選択 |
| Time | 時刻 | 50 | ★ | 時刻（HH:mm）の入力 |
| TimeCalculate | 時刻計算 | 55 | | 時刻同士の差分を自動計算 |

**よくある使い分け:**
- 実施日・記入日 → **Date**
- 期間指定（開始〜終了） → **CalendarDate**
- 開始時刻・終了時刻 → **Time**
- 作業時間（終了 - 開始） → **TimeCalculate**

## 選択・チェック

選択肢やチェックボックスに関するタイプです。

| タイプ名 | 日本語名 | 値 | AI | 用途 |
|----------|----------|-----|-----|------|
| Check | チェック | 90 | ★ | 単一のチェックボックス |
| MultipleChoiceNumber | トグル選択 | 123 | | トグル切替式の数値選択 |
| MCNCalculate | トグル集計 | 124 | | トグル選択の集計計算 |
| Select | 単一選択 | 70 | ★ | ドロップダウン等で1つを選択 |
| MultiSelect | 複数選択 | 80 | | ドロップダウン等で複数を選択 |
| SelectMaster | マスター選択 | 125 | | マスターデータから選択 |

**よくある使い分け:**
- 合否・確認チェック → **Check**
- 区分・ステータス（1つ選択） → **Select**
- 該当項目を複数チェック → **MultiSelect**
- マスターテーブルから検索して選択 → **SelectMaster**
